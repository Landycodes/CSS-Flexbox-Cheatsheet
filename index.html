CSS Cheatsheet

Here's a Cheatsheet of some common use cases for CSS Flexbox! Click a box and copy/paste into your code, easy peasy!

Flexbox Row

Use these properties to create a Flexbox row Layout.
 .row {
     display: flex;
     flex-direction: row;
     flex-wrap: wrap;
   }

Flexbox Column

Use this to create a Flexbox column layout.
 .column {
     display: flex;
     flex-direction: column;
 }

CSS Grid Layout

This will create a 12 column row layout with CSS grid.
 .grid {
     display: grid;
     width: 100%;
     grid-template-columns:
 repeat(12, 1fr);
 }

Linear Gradients

Creates a linear-gradient backround from top to bottom.
 .linear-gradient-background {
     backround-image: linear-gradient(
         rgba(232, 102, 236, 0.3) 0%,
         rgba(232, 102, 236, 0.6) 100%
     );
 }

Box Transition Glow

Uses Transition and box shadows to glow on hover.
 .code-card .card-header {
     border-radius: 8px;
     Transition: all 0.5s ease-in-out;
 }

 .code-card:hover,
 .code-card:hover .card-header {
     box-shadow: inset 0px 0px 8px
 rgba(232, 102, 236, 1), 0 0 15px
 rgba(232, 102, 236, 0.3);
 }

Overlay Card with Title

Use position properties and negative margins to raise elements higher then there natural starting point.
 .card-header {
     position: relative;
     margin-top: -20px;
 }
